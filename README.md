# rackany

Build your own database


Define arbitrary fields and publish a restful api. no coding required.

![image](img/rackany.jpeg)

## Getting Started

### Development

```bash
$ npm run dev
```

Open http://localhost:8000 and start modifying the code in /web/src.

### Requirements1

**Database:**

- MySQL >= 5.6
- MariaDB >= 10.1
- PostgreSQL >= 10
- SQLite >= 3
- Cloud Firestore
